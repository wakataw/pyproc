Usage Mode: MCP Server

The MCP server allows LLM clients (such as Claude Desktop or Cursor) to call PyProc tools. It supports two transportation protocols: stdio (local processes) and HTTP (SSE stream servers).

1. Generating Configuration

To easily get configuration lines for your clients, use the built-in generator:

bash 
# Output stdio client config to stdout
pyproc mcp --generate-config

# Output HTTP client config to stdout
pyproc mcp --interface http --port 9090 --generate-config

# Write to file directly
pyproc mcp --generate-config mcp-config.json

2. Claude Desktop Integration

Add the configuration blocks below to your Claude Desktop config file (located at ~/Library/Application Support/Claude/claude_desktop_config.json on macOS or %APPDATA%\Claude\claude_desktop_config.json on Windows):

json 
{
  "mcpServers": {
    "pyproc": {
      "command": "pyproc-mcp",
      "env": {
        "PYPROC_TIMEOUT": "30"
      }
    }
  }
}

3. Cursor Integration

In Cursor Settings → Features → MCP:

  1. Click + Add New MCP Server.
  2. Set Name to pyproc.
  3. Set Type to command.
  4. Set Command to: pyproc-mcp.
