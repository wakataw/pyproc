MCP SERVER FOR LLM CLIENTS

Real-Time Indonesian Procurement Data for AI Agents

PyProc MCP turns public SPSE/Inaproc procurement data into standard MCP tools, libraries, and CLI subcommands. Bring real-time bidding intelligence straight to Claude, Cursor, and custom agents.

Enable Procurement Intelligence

Harness real-time API bindings and tools to discover opportunities, optimize competition, and automate data retrieval.

Opportunities Discovery

Search active tender and non-tender packages across hundreds of LPSE hosts. Filter by keyword, category, budget year, and instansi instantly.

Competitive Intelligence

Inspect bidders lists, read participant details, and evaluate winners prices to uncover competitor pricing margins and winning strategies.

Vendor & Contractor Research

Look up specific vendor names to find their historical government wins, target location values, and tender qualification achievements.

Timeline & Schedule Tracking

Track real-time bidding schedule phases from initial announcement to contract signature. Monitor timeline changes and deadlines automatically.

Automated Bulk Export

Download whole SPSE listings for reporting and databases. The CLI features automated resuming and writes clean CSV or JSON structures.

AI Agent Integration

Directly connect your Cursor, Continue, or Claude Desktop environments. Enable LLMs to run complex checks, matching queries on the fly.

Explore Advanced Functions

PyProc provides robust fallback mechanics like the LKPP ISB Satu Data API, SQLite full-text index synchronization, custom rate-limiting, and TLS certificate bypass controls.

python Python SDK 
from pyproc import Lpse

# Context manager handles connections cleanly
with Lpse('kemenkeu') as lpse:
    detail = lpse.detil_paket_tender('10080116000')
    detail.get_all_detil()
    print("Winner:", detail.get_pemenang())
